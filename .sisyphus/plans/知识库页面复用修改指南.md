# 知识库管理页面复用现有页面 - 修改指南

## 修改目标

将知识库管理中心的"药品管理"、"诊疗项目管理"、"医院管理"改为跳转到现有的"医保目录管理"和"医院信息管理"页面。

## 需要修改的文件

### 1. `components/knowledge/KnowledgeManagementPage.tsx`

**修改步骤：**

```tsx
interface KnowledgeManagementPageProps {
  onViewChange?: (view: string) => void;
}

const KnowledgeManagementPage: React.FC<KnowledgeManagementPageProps> = ({ onViewChange }) => {
  // ...
```

## 完整代码参考

### KnowledgeManagementPage.tsx 完整修改后代码

```tsx
import React, { useState } from 'react';

interface KnowledgeManagementPageProps {
  onViewChange?: (view: string) => void;
}

const KnowledgeManagementPage: React.FC<KnowledgeManagementPageProps> = ({ onViewChange }) => {
  const [currentTab, setCurrentTab] = useState('aliases');

  // 外部链接（跳转到现有页面）
  const externalLinks = [
    { 
      id: 'drugs', 
      label: '药品目录管理', 
      icon: '💊', 
      view: 'medical_catalog_management',
      description: '管理医保药品目录，包含甲类/乙类/丙类药品'
    },
    { 
      id: 'services', 
      label: '诊疗项目管理', 
      icon: '🩺', 
      view: 'medical_catalog_management',
      description: '管理医保诊疗项目目录'
    },
    { 
      id: 'hospitals', 
      label: '医院信息管理', 
      icon: '🏨', 
      view: 'hospital_management',
      description: '管理医院基本信息和等级'
    },
  ];

  // 内部管理的Tab（在知识库页面内显示）
  const internalTabs = [
    { id: 'aliases', label: '别名映射', icon: '🔗' },
    { id: 'diseases', label: '疾病管理', icon: '🏥' },
    { id: 'rules', label: '规则管理', icon: '📋' },
    { id: 'relationships', label: '关系管理', icon: '🕸️' },
    { id: 'versions', label: '版本管理', icon: '📚' },
  ];

  const handleExternalLinkClick = (view: string) => {
    if (onViewChange) {
      onViewChange(view);
    }
  };

  const renderContent = () => {
    switch (currentTab) {
      case 'diseases':
        return <DiseaseManagement />;
      case 'aliases':
        return <AliasMappingPage />;
      case 'rules':
        return <RuleManagementPage />;
      case 'relationships':
        return <RelationshipManagementPage />;
      case 'versions':
        return <VersionManagementPage />;
      default:
        return <div className="p-8 text-center text-gray-500">请选择管理模块</div>;
    }
  };

  return (
    <div className="p-6">
      <div className="mb-6">
        <h1 className="text-2xl font-bold">知识库管理中心</h1>
        <p className="text-gray-600">管理医疗知识库主数据、规则和关系图谱</p>
      </div>

      {/* 外部链接卡片 */}
      <div className="mb-6">
        <h2 className="text-lg font-semibold mb-3 text-gray-700">主数据管理（链接到现有系统）</h2>
        <div className="grid grid-cols-1 md:grid-cols-3 gap-4">
          {externalLinks.map(link => (
            <button
              key={link.id}
              onClick={() => handleExternalLinkClick(link.view)}
              className="p-4 bg-white rounded-lg shadow-sm border border-gray-200 hover:border-blue-400 hover:shadow-md transition-all text-left group"
            >
              <div className="flex items-center gap-3 mb-2">
                <span className="text-2xl">{link.icon}</span>
                <span className="font-medium text-gray-900 group-hover:text-blue-600">
                  {link.label}
                </span>
                <svg className="w-4 h-4 text-gray-400 ml-auto group-hover:text-blue-500" fill="none" stroke="currentColor" viewBox="0 0 24 24">
                  <path strokeLinecap="round" strokeLinejoin="round" strokeWidth={2} d="M9 5l7 7-7 7" />
                </svg>
              </div>
              <p className="text-sm text-gray-500">{link.description}</p>
            </button>
          ))}
        </div>
      </div>

      {/* 内部分类Tab */}
      <div className="mb-6">
        <h2 className="text-lg font-semibold mb-3 text-gray-700">知识库特有功能</h2>
        <div className="flex gap-4 border-b mb-4">
          {internalTabs.map(tab => (
            <button
              key={tab.id}
              onClick={() => setCurrentTab(tab.id)}
              className={`px-4 py-2 flex items-center gap-2 border-b-2 transition-colors ${
                currentTab === tab.id
                  ? 'border-blue-500 text-blue-600'
                  : 'border-transparent text-gray-600 hover:text-gray-900'
              }`}
            >
              <span>{tab.icon}</span>
              <span>{tab.label}</span>
            </button>
          ))}
        </div>
      </div>

      <div className="bg-white rounded-lg shadow-sm border border-gray-200 min-h-[400px]">
        {renderContent()}
      </div>
    </div>
  );
};

// 保留内部功能组件（DiseaseManagement, AliasMappingPage, RuleManagementPage, RelationshipManagementPage, VersionManagementPage）

export default KnowledgeManagementPage;
```

## 验证步骤

1. 保存修改后的文件
2. 启动开发服务器：`npm run dev`
3. 访问知识库管理中心（理赔管理 → 知识库管理）
4. 点击"药品目录管理"卡片，应跳转到医保目录管理页面
5. 点击"诊疗项目管理"卡片，应跳转到医保目录管理页面
6. 点击"医院信息管理"卡片，应跳转到医院信息管理页面
7. 点击下方的Tab（别名映射、疾病管理等），应在当前页面内显示

## 优点

1. **复用现有功能**：充分利用已有的医保目录管理和医院信息管理页面
2. **工作量小**：只需要修改页面跳转逻辑，不需要重写功能
3. **体验一致**：用户在不同页面间跳转，功能完整
4. **架构清晰**：主数据管理使用现有系统，知识库特有功能独立开发
